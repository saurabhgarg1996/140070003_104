Title of the Project : Library Helpmate Firebird Bird V Bot
Instructions :
1.Follow the instructions at http://kevinhughes.ca/tutorials/opencv-install-on-windows-with-codeblocks-and-mingw/
 to install and integrate OpenCV with Code::Blocks.
2.Install X-CTU http://www.digi.com/products/wireless-wired-embedded-solutions/zigbee-rf-modules/xctu
3.Hardware Requirements : Fire Bird V, a pair of Xbee with an adapter, 3 Sharp Sensors, webcam, servo motor and a gripper
4.Open source code using Code::Blocks and, build and run Library_Database.cpp and Capture.cpp .
5.Passkey to Library Database is "cs101".
6.Open Embedded_c.c in Atmel Studio, in Project column set the frequency as 14745600 in configuration option.  
7.Build and run the Embedded_c.c code after including lcd.h header file to that directory.
8.Load the hex file generated in default option in that directory on Fire Bird V using AVR Bootloader.
9.Configure your Xbee using X-CTU. Refer to youtube link : https://www.youtube.com/watch?v=Boy5asltEJw
